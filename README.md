# ChiselBox ( Adnroid )
- WS tunnel for v2ray using [chisel library](https://github.com/jpillora/chisel) 
 - قابل اجرا بر روی اندروید 10 و بالاتر
 - نیازمند دامین ریجستر شده روی کلادفلیر
